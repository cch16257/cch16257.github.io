Python中文件、模块的意义与导入方式

Python中的模块

  1. 模块的含义

python中的一个扩展名为.py文件就是一个模块。

一个模块中包含：类、语句、函数等

  1. 模块的好处
  • 方便其它程序和脚本的导入并使用
  • 避免函数名和变量名冲突
  • 提高代码的可维护性
  • 提高代码的可重用性
  1. 模块的导入

import moduleA

或

from moduleA import functionA

Python中的包

  1. 包的含义

包是一个分层次的目录结构，它将一组功能相近的模块组织在一个目录下

  1. 包的作用
    • 代码规范
    • 避免模块名称冲突
  2. 包与目录的区别
    • 包含_init__.py文件的目录称为包·_
    • 目录里通常不包含_init__.py文件
  3. 包的导入

import moduleA

import packageA.moduleA

from packageA.moduleA import functionA

import 与 from…import…的区别

  1. import只能后跟包或模块，而不能跟函数等
  2. from…import…中的import 可以后跟函数、类、变量等
