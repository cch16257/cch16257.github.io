Anaconda下TensorFlow-GPU的安装

本篇文章是在Anaconda的虚拟环境下安装CUDA、cuDNN和TensorFlow-GPU，创建虚拟环境省略。

一、查看本机显卡驱动所能支持的CUDA最高版本

cmd命令行输入,即可查看本机显卡驱动所能支持的CUDA最高版本：

二、选择自己想安装的Tensor Flow版本，下载对应的CUDA包、cuDNN包和Tensor Flow。

1. 选择自己想安装的Tensor Flow版本

去tensorflow官网查看Tensor Flow版本，和其对应的CUDA包与cuDNN包：Tensor Flow版本。而我想下载的版本是：tensorflow_gpu-2.6.0，所对应的是cuDNN8.1版本与CUDA11.2版本，并且我所想下载的CUDA版本也未超过我本机所能下载的最高版本。

如果想查看cuDNN与CUDA两者的版本是否对应，可访问：cuDNN Archive，需先Login和Join。

2. 在conda环境中安装cuda

首先创建并激活conda虚拟环境。

然后查看对应版本的cuda下载地址，下载安装包到本地，在虚拟环境中安装。

3. 在conda环境中安装cuDNN

激活conda虚拟环境下，查看对应版本的cuDNN下载地址，下载安装包到本地，在虚拟环境中安装。

4. pip安装tensorflow-gpu

激活conda虚拟环境下

三、测试

1. 测试是否安装成功

创建python文件：tensorflow_install_test.py

2. 测试GPU能否使用

创建python文件：tensorflow-gpu_install_test.py
